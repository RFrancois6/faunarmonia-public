# 🐔 Faunarmonia — A Rhythm Auto-Battler

> **Patapon × Clash Royale, in a Game of Thrones meets Rabbids universe.**
>
> An isometric rhythm auto-battler where chickens and foxes face off to the sound of the music.

🎮 **Game page (private access)**: [rfrancois6.itch.io/faunarmonia](https://rfrancois6.itch.io/faunarmonia?password=demo-recruiter)
📌 **Status**: in active development — Kickstarter campaign in preparation.

---

## The concept

Pick a faction — chickens or foxes — and complete objectives (survival, capture the flag, more to come) by summoning units to the rhythm of the music. Three unit types in a rock-paper-scissors balance: archer, assassin, knight. Miss the beat, miss the summon.

## Influences

- **Patapon** — the rhythmic summon mechanic
- **Clash Royale** — the auto-battler loop and rock-paper-scissors balance
- **Game of Thrones** — medieval atmosphere and faction rivalry
- **The Rabbids** — the offbeat, cartoony tone running throughout

## Tech

- **Engine**: Unreal Engine 5
- **Language**: Blueprint (gameplay architecture + level generator)
- **Target platforms**: Steam (priority), mobile if funding allows
